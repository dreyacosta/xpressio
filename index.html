xpressio

Just a small and fast tool to design your website.

A little exercise to try Stylus
Download View on Github
Version 0.3

Simple to use and customize

Don't worry about your CSS knowledge, just set your settings, colors and build your xpressio theme in less than 5 minutes.

CSS3 Powered

CSS3 it's fun, with powerful features that makes it easy to create a Web Site.

Built with Stylus

Preprocesors are great tools providing scalable and maintainable CSS.

Lightweight

High performance Web Sites are a plus so Xpressio is lightweight. 20kb max!

Feel free to change settings

Just edit your __variables.styl and compile xpressio.styl.

// Font settings

font_base = Lato
font_base_size = 16px
font_base_weight = 300

font_book_weight = 300
font_bold_weight = 700
font_normal_weight = 400
font_thin_weight = 100

// Headings (h1, h2, h3, ...)

heading_font_weight = 300
heading_line_height = 1.2em
heading_margin = 0 0 0 0
// Logo, header and site width container

logo_font_size = 1.8em
header_line_height = 3em
site_width = 1024px

// Paragraph (p), margins and padding settings

paragraph_line_height = 1.5em
paragraph_margin = 0 0 10px 0
paragraph_right_left = 0

margin_small = 5px
margin_medium = 20px
// Grid

column_width = 100px
gutter_width = 10px
columns = 12 // max 16
total_width = 100%

// More colors

mysauce1 = #D2B48C

// Colors degradation

light = 90%
