xpressio

Just a small and fast tool to design your website.

Download View on Github
Version 0.1

Simple to use and customize

Don't worry about your CSS knowledge, just set your settings, colors and build your xpressio theme in less than 5 minutes.

CSS3 Powered

CSS3 it's fun, with powerful features that makes it easy to create a Web Site.

Built with Stylus

Preprocesors are great tools providing scalable and maintainable CSS.

Lightweight

High performance Web Sites are a plus so Xpressio is lightweight. Only 12kb!

Feel free to change settings

Just edit your xpressio.theme.styl and compile.

// General settings

@import url(http://fonts.googleapis.com/css?family=Lato:100,300,400,700)	

font = Lato
base_font_size = 16px
base_font_weight = 300
heading_font_weight = 300
heading_line_height = 1.2em
site_width = 1024px

paragraph_line_height = 1.5em
paragraph_bottom = 10px
paragraph_right_left = 0

margin_top_small = 20px
margin_top_big = 40px

margin_bottom_small = 20px
margin_bottom_big = 40px

padding_small = 20px
padding_big = 40px

border_radius_small = 5px
border_radius_big = 15px
// Grid

column_width = 100px
gutter_width = 10px
columns = 12 // max 16

total_width = 100%

// More colors

mySauce1 = #D2B48C
